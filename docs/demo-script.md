# Demo Script

Target: 2:30 to 2:45.

1. TraceCue is not an AI SOP generator.
2. It turns messy handoff notes into reviewable guide cards.
3. Each card shows source references.
4. Risky claims are flagged.
5. Publish Gate blocks unsupported or unreviewed steps.
6. ProcedureLedger records the lifecycle.
7. Feedback becomes a v1.1 revision proposal.

Closing line:

> TraceCue makes procedure generation publishable by combining source grounding, human review, and replayable governance.
